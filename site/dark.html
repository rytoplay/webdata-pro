Features
Everything you need, nothing you don’t

Built by a web developer who got tired of rebuilding the same back end for every client.

🤖

AI app builder

Works with Ollama (free, local), Claude, GPT-4, or Gemini. You pick the model; you own every piece of the output.

✏️

Visual template editor

Plain HTML with ${table.field} tokens. Images, distance, aggregates, sorting — all built-in token helpers.

👥

Member portal & groups

Login system with groups, per-group access control, and per-group home views. Each member sees only what they’re allowed to.

🔗

Embeddable anywhere

One <script> tag on any website — WordPress, Squarespace, raw HTML. The widget inherits your existing CSS.

📋

Public forms

Let anonymous visitors submit records without logging in. Optional Google reCAPTCHA v2 keeps bots out.

📬

Email notifications

Alert admins when records change — immediately or as a daily digest. Configured via SMTP in the admin panel.

🗄️

SQLite or MySQL

SQLite requires zero setup. Swap to MySQL for multi-user production when you’re ready. The admin works identically.

⚡

No build step

npm run dev and you’re in. TypeScript compiles on demand. No Webpack, no Vite, no waiting.

You’re always in control
The AI generates it.

There is no magic layer between you and the output. Change one line or rebuild from scratch — it’s all yours.

Get started
Five commands. No cloud account needed.

No Docker, no credit card, no deployment pipeline. Just Node.js and a terminal.

zsh — webdata-pro 
# Clone and install
$ git clone https://github.com/rytoplay/webdata-pro.git
$ cd webdata-pro && npm install

# Configure and start
$ cp .env.example .env
$ npm run dev

# Open http://localhost:3456/admin
