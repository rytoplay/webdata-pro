How it works
From blank page to working app in three steps

No back-end code to write. No database schema to design. Just describe what you need.

1

Describe your app

Type a plain-English description — “a searchable database of pets for adoption with name, breed, and temperament.” The AI creates your tables, fields, and sample records in under a minute. Or skip the AI and set everything up by hand.

2

Preview immediately

A working, searchable application appears right in your browser. Search, filter, pagination, and detail views are all wired up automatically. Customize the HTML templates however you like — change one line or rebuild from scratch.

3

Embed anywhere

Drop one <script> tag into any web page — WordPress, Squarespace, raw HTML, it doesn’t matter. The widget inherits your existing CSS. Your data stays on your server, under your control.

Features
Everything you need, nothing you don’t

Built by a web developer who got tired of rebuilding the same back end for every client.

🤖

AI app builder

Works with Ollama (free, local), Claude, GPT-4, or Gemini. You pick the model; you keep every piece of output it generates.

✏️

Visual template editor

Plain HTML with ${table.field} tokens. Sorting, images, distance calculations, aggregates — all built-in token helpers.

👥

Member portal & groups

Login system with groups, per-group access control, and per-group home views. Each member sees only what they’re allowed to see.

🔗

Embeddable anywhere

One <script> tag on any website. Works with WordPress, Squarespace, or a plain HTML file on a flash drive.

📋

Public forms

Let anonymous visitors submit records without logging in. Optional Google reCAPTCHA v2 keeps spam bots out.

📬

Email notifications

Alert admins when records are created or changed — immediately or as a daily digest. Configure via SMTP in the admin panel.

🗄️

SQLite or MySQL

SQLite requires zero setup — just start the server. Swap to MySQL for multi-user production when you’re ready.

⚡

No build step

npm run dev and you’re in. TypeScript compiles on demand. No Webpack, no Vite, no waiting.

The philosophy
AI-assisted. Designer-controlled.
“The AI handles the tedious structural work — your tables, fields, search forms, and starter templates. Everything it produces is plain HTML and SQL that you can read, edit, and fully control. There is no magic layer between you and the output.”

Other AI builders lock you into their styling system, their embed limitations, their notification pipeline. When you need to change something they didn’t anticipate, you’re stuck. Webdata Pro gets you to a working first draft in minutes — then gets out of your way so the designer can take it the rest of the way on their own terms.

Get started
Five commands. No cloud account needed.

No Docker, no credit card, no deployment pipeline. Just Node.js and a terminal.

  • Node.js 18 or higher — download from nodejs.org
  • SQLite database included, nothing else to install
  • Admin panel opens at localhost:3456/admin
  • AI builder is optional — build apps by hand if you prefer
  • Move to MySQL or a cloud host whenever you’re ready
